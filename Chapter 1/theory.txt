What is Emmet?
Emmet is a plugin for ide's to auto complete and autofill a code snippet. Allows for high speed coding.

Difference between a Library and Framework?
Library can be imported and used on a specific section of your code, whereas framework is applied to the whole code. React is a Library.

What is CDN? Why do we use it?
CDN is Content Delivery Network. It is a group of server spread across the world to deliver web content to users quickly.

Why is React known as React?
React is called React because it reacts to changes in data without reloading the entire page. It is Responsive and Efficient. Created by Facebook.

What is crossorigin in script tag?
Cross-origin resource sharing (CORS) is a security mechanism that allows web applications to access resources from other domains, such as a CDN. It prevents malicious website from accessing data without explicit permissions.

What is difference between React and ReactDOM?
React is a JavaScript library for building user interfaces, while ReactDOM is a separate package that provides DOM-specific methods for managing React components.

What is difference between react.development.js and react.production.js files via CDN?
Use react.development.js when developing your application. It provides valuable feedback and debugging tools.
Use react.production.js when deploying your application to a live environment. It ensures optimal performance and security.

What is async and defer?
Attributes that instruct the browser to download and execute scripts without blocking the page's rendering.
Async => HTML Parsing, Script Loading --> HTML Paused, Script Execution --> HTML Parsing
Defer => HTML Parsing, Script Loading --> HTML Script Execution at end
